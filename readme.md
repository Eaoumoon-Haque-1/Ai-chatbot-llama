# 🚀 AI Document Chatbot – Retrieval Augmented Generation (RAG) Platform

## 📌 Overview

AI Document Chatbot is a full-stack AI application that allows users to upload documents such as PDFs and text files, ask natural language questions, generate summaries, and interact with their own private knowledge base.

Unlike standard chatbots that answer only from pretrained knowledge, this system uses **Retrieval-Augmented Generation (RAG)** to retrieve relevant information directly from uploaded documents and then generate accurate, context-aware responses.

This project demonstrates practical software engineering and modern AI integration using:

- Large Language Models (LLMs)
- Vector Databases
- Semantic Search
- API Development
- Custom Frontend UI
- Multi-model AI Switching
- Document Processing Pipelines

---

# 🎯 Why I Built This

Many users and organizations have large numbers of PDFs, reports, notes, CVs, assignments, research papers, and internal documents.

Searching manually is slow.

This platform solves that problem by allowing users to:

✅ Upload files instantly  
✅ Ask questions naturally  
✅ Get summaries in seconds  
✅ Extract key information quickly  
✅ Switch between multiple AI models  
✅ Use AI on their own documents
